Applications

Runtime apps built on AEAOS Solutions (L5)
Composable, versioned, governance-bound runtime apps deployed on top of AEAOS Solutions — the L5 layer of the 7-layer lifecycle stack.

// Application Registry

Every application is the true-source in registry/application.yaml and resolved by ApplicationManager at startup (Registry-first, P1). Applications advance through the environment promotion chain (P5) and are bound to their parent Solution + Kernel Capabilities.

// Manage via CLI

Manage applications via CLI:

aeaos app register|list|info|search|run
